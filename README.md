# Scores
Service for evaluating user bets given a match's final result. 

## Payload format for the results endpoint
{  
	"results":{
        "result":{  
            "matchId":""+matchId+"",  
            "competitionId":"compId100",  
            "homeTeamName":"team1",
            "awayTeamName":"team2",
            "homeTeamGoals":1,  
            "awayTeamGoals":1  
        }  
    },  
    "token":"ashdj"
}
